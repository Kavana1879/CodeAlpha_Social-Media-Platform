# StudySphere — AI Powered Student Social Media Platform

StudySphere is a futuristic student social media platform built using Django, HTML, CSS, and JavaScript.

## Features

- User Authentication (Login/Register/Logout)
- Student Profiles
- Posts & Comments
- Like System
- Follow/Unfollow System
- Student Skill Badges
- AI Notes Summary
- Notifications
- Responsive Soft Neon UI
- Animated Splash Screen

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python Django

### Database
- SQLite

## Installation

### Clone Repository

```bash
git clone <your-github-repo-link>
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows
```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install django pillow
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```
